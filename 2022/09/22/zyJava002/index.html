Scanner类

java.util.Scanner 是 Java5 的新特征，
我们可以通过 Scanner 类来获取用户的输入

(使用next方法)例如Zy.java文件代码：

import java.util.Scanner; //导入Scanner的路径

　　 public class Zy.java{
　　　　 public static void main(String[] args) {
　　　　　　 Scanner input=new Scanner(System.in); 创建Scanner对象,从键盘输入数据
　　　　　　 int b = input.nextInt(); 使用Scanner对象input,调用nextInt()方法,在键盘输入一个int型数据

　　　　　　 input.close(); 关闭Scanner对象input
　　 }
}
